Introduction

Vue.js is an open-source JavaScript framework for building user interfaces.[4] Integration into projects that use other JavaScript libraries is simplified with Vue because it is designed to be incrementally adoptable. Vue can also function as a web application framework capable of powering advanced single-page applications.
